# DIS5K Object Classification

**Модель UNet++** для сегментации объектов архитектуры и артефактов. [conversation_history:6]

## Данные
- **Обучающая выборка**: 3000 изображений  
- **Валидационная выборка**: 300 изображений  
- **Датасет**: DIS5K (20+ классов: балконы, мосты, трещины, заборы и др.)

## Результаты
MSE: 700 на валидационной выборке
Архитектура: UNet++ (PyTorch, Segmentation Models)
Размер изображений: 1024×1024

## Стек
PyTorch | Segmentation Models PyTorch | OpenCV | NumPy
